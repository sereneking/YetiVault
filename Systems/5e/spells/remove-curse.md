# Remove Curse
%%-- Embedded content starts on the next line. --%%
*3rd-level, Abjuration*  

- **Casting time:** 1 action
- **Range:** Touch
- **Components:** V, S
- **Duration:** Instantaneous

At your touch, all curses affecting one creature or object end. If the object is a cursed magic item, its curse remains, but the spell breaks its owner's attunement to the object so it can be removed or discarded.

**Classes**: [Cleric](/Systems/5e/classes/cleric.md), [Fighter (Eldritch Knight)](/Systems/5e/classes/fighter-eldritch-knight.md), [Paladin](/Systems/5e/classes/paladin.md), [Rogue (Arcane Trickster)](/Systems/5e/classes/rogue-arcane-trickster.md), [Sorcerer (Divine Soul)](/Systems/5e/classes/sorcerer-divine-soul-xge.md), [Warlock](/Systems/5e/classes/warlock.md), [Wizard](/Systems/5e/classes/wizard.md)
